You already solved it using Set, which is the cleanest modern approach.
If you want to solve it using array functions only (like filter, reduce, etc.), here are a few solid alternatives.

⭐ Using filter
This keeps the first occurrence of each value:

js
const arr = [1, 2, 2, 3, 4, 4, 4, 5];

const removeDuplicates = arr =>
  arr.filter((value, index) => arr.indexOf(value) === index);

console.log(removeDuplicates(arr));
How it works:

indexOf(value) returns the first index where the value appears

If that matches the current index, it's the first occurrence → keep it

⭐ Using reduce
This builds a new array step by step:

js
const removeDuplicates = arr =>
  arr.reduce((acc, value) => {
    if (!acc.includes(value)) acc.push(value);
    return acc;
  }, []);

console.log(removeDuplicates(arr));
⭐ Using forEach (still an array method)
Simple and readable:

js
const removeDuplicates = arr => {
  const result = [];
  arr.forEach(v => {
    if (!result.includes(v)) result.push(v);
  });
  return result;
};

console.log(removeDuplicates(arr));
